# Proyecto de Demostración de Algoritmos en Python

Este proyecto implementa y muestra el funcionamiento de varios algoritmos clásicos
de optimización y búsqueda, utilizando Python.

Incluye:

- Problema de Asignación 1 a 1  
- Problema de la Mochila  
- Problema del Vendedor Viajero (TSP)  
- Problema de Distribución de Recursos  
- Y adicionalmente, medición de tiempo en milisegundos

---

## Ejecución

Ejecute el programa principal:

```bash
demo.py
```
